SCP판매예상 + 비재고 + 추가건 + 충족률보정 → 일투입 산출

⚠ 라인 미매핑 품목 — 라인을 지정해주세요

아래 품목들은 재고리스트에 라인 정보가 없습니다. 라인을 지정하면 분석에 포함됩니다.

📖 SCP 라인별 부하 분석 사용 설명서

📁 파일 준비 방법

① SCP 운영계획
SCM팀에서 받은 의자 SCP 운영계획 엑셀 파일을 그대로 업로드합니다.
예: 시디즈 입고가 기준 5-7월 의자 SCP 운영 계획.xlsx
② 재고품목 리스트
ERP → 일반 생산계획관리 창에서 조회 후 엑셀로 변환하여 업로드합니다.
1. ERP → 생산관리 → 일반 생산계획관리 접속
2. 해당 월 기준으로 전체 조회
3. 상단 [엑셀저장] 클릭 → 파일 저장
4. 저장된 grd_list00_XXXXXX.xls 업로드
포장라인·현재고·목표재고 정보가 포함된 파일
③ 생산계획 (투입)
ERP → 일반 생산계획정보 창에서 전월 한 달 전체 조회 후 엑셀 변환하여 업로드합니다.
1. ERP → 생산관리 → 일반 생산계획정보 접속
2. 조회기간: 전월 1일 ~ 말일 입력
3. 전체 조회 → [엑셀저장] 클릭
4. 저장된 Grid00_XXXXXX.xls 업로드
비재고 비율 계산에 사용 (SCP에 없는 품목 = 비재고)

⚙ 설정 항목 설명

분석 대상 월
분석할 월을 선택합니다. SCP 파일에서 해당 월의 판매예상량과 목표재고 열을 자동으로 찾습니다.
전월 / 당월 영업일
일투입 계산에 사용되는 영업일 수입니다. 달력 기준으로 직접 입력합니다.
전월 투입(실적) ÷ 전월 영업일 = 전월 일투입
충족률 기준(%)
현재고가 목표재고 대비 이 비율 미만이면 부족분을 일투입에 추가합니다.
기본값 90% → 충족률 90% 미만 라인만 보정
품목 이관 규칙
재고리스트에 등록된 라인과 실제 포장라인이 다를 때 사용합니다.
예) 내수 6200 코드 → T40-2_F 대신 벌크 라인으로 집계
추가건
SCP에 없는 물량(대량 수주건, 신규 출시 초도물량 등)을 수기로 추가합니다.
예) TC13 퍼시스 대량건 +1,500 / 만타 가죽라인 +500
📌 일투입 계산식
전월 일투입 = 전월 전체 투입 ÷ 전월 영업일
당월 필요생산 = 목표입고 + 당월목표재고 − 현재고
당월 일투입 = (목표입고 + 비재고 유지 + 추가건) ÷ 당월 영업일

기본 설정

품목 이관 규칙

재고리스트 라인과 실제 포장라인이 다른 품목을 지정합니다.

재고 강제 지정

SCP에 미등록이라 비재고로 분류되지만, 실제로는 재고로 운영하는 품목을 지정합니다.

추가건 / 신규 제품 수기 입력

대량건, 신규 출시, SCP 미반영 물량 등을 수기로 추가합니다.

라인별 CAPA (일 생산능력)

각 라인의 일일 최대 생산능력을 입력합니다. 0이면 무제한으로 간주합니다.

비재고 라인 수기 금액 (월간)

전량 비재고로 운영되는 라인의 월간 생산 예상 금액을 입력합니다.
